Load Files

  • Create a folder to connect to files: Folder  and select the file location folder - Name your connection

After creating connection, select data files from connection folder

  • Select comma delimiter
  • Click insert script

  • Repeat for all excel files
  • Name tables corresponding to their data: [US_state_vaccinations], [2021VAERSDATA],  [2021VAERSSYMPTOMS],  [2021VAERSVAX], [Population_by_States], [People_vaccinated_US].
